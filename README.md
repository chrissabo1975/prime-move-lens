## The Idea

Everything follows the same five-stage cycle.

A thought. A conversation. A tattoo. A surgery. A civilization. Making eggs in the morning.

All of it runs on the same code:

|Stage           |What It Is                                      |
|----------------|------------------------------------------------|
|**Split**       |A distinction appears where there was none      |
|**Tension**     |Imbalance, gradient, desire, pressure           |
|**Failed Merge**|Attempted resolution that leaves residue        |
|**Scar**        |The residue — persistent structure, memory, form|
|**Decay**       |Relaxation, release, enabling the next cycle    |

This paper is not a proof. It is a manual.

You do not need math. You do not need philosophy. You only need the willingness to look.

-----

## What’s Inside

The paper builds from the smallest scale outward:

- **Your own thoughts** — the cycle running right now, as you read this
- **Conversation** — every conversation produces a mutual understanding. That’s all it does. That’s enough.
- **The tattoo** — one of humanity’s oldest practices of deliberate scar-making, mapped stage by stage
- **Surgery** — a necessary scar. Same process, different intention.
- **Your phone** — engineered to maximize Splits and minimize Scar formation. The lens makes that visible.
- **Making eggs** — the most important example. Every action is a Split initiating a run of the code. The code stacks.
- **Your existence** — how you came to be, mapped to the five stages. You are a scar. This is not a metaphor.
- **Contemporary society** — where we are in the cycle, which pathway we’re on, and what the lens suggests (not prescribes)

-----

## The Four Axioms Behind It

You only need these:

- **Axiom 0:** Nothing cannot persist. Something must emerge.
- **Axiom 1:** Before any structure, there is undifferentiated unity.
- **Axiom 2:** Distinction is necessary. Without it, nothing happens.
- **Axiom 3:** Distinction cannot be perfectly erased. It leaves residue.

From these four axioms, the five-stage cycle follows necessarily. Everything else is elaboration.
